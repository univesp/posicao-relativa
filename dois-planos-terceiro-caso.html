Posição relativa entre dois planos

Para dois planos há três possibilidades, que são: planos paralelos coincidentes (iguais), planos paralelos distintos e planos transversais.

A seguir iremos ver esses casos e como podemos identificá-los algebricamente.

2º Caso

3º Caso

Planos transversais

Planos transversais

Posição relativa

Neste caso, os planos se intersectam em uma reta.

Como identificar

Para verificar se os planos são transversais, basta analisar a posição relativa dos vetores normais aos planos, para isso faremos o produto vetorial entre eles, se o resultado for diferente de 0, então os vetores são linearmente independentes (LI) e, portanto, os planos são transversais
