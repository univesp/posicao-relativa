Posição relativa entre dois planos

Para dois planos há três possibilidades, que são: planos paralelos coincidentes (iguais), planos paralelos distintos e planos transversais.

A seguir iremos ver esses casos e como podemos identificá-los algebricamente.

Planos paralelos coincidentes

Posição relativa

Neste caso, os planos são iguais, logo, ocupam o mesmo lugar no espaço.

Como identificar

1º passo

Para verificar se os planos são paralelos coincidentes, primeiro devemos analisar a posição relativa dos vetores normais aos planos, para isso, faremos o produto vetorial entre eles, se o resultado for igual a 0 então os vetores são linearmente dependentes (LD) e, portanto, paralelos.

2º passo

Em seguida, visto que os vetores normais dos planos são LD, devemos pegar um ponto pertencente a um dos planos e verificar se o ponto também pertence ao outro plano, se sim, os planos são paralelos coincidentes.
