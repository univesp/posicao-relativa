Neste caso, as retas são paralelas distintas, logo, nunca se encontram.

Como identificar

1º passo

Para identificar se duas retas são paralelas distintas, primeiro precisamos verificar se os vetores diretores das retas são linearmente dependentes (LD), pois, neste caso, ambas as retas possuem a mesma direção no plano, sendo assim, são retas paralelas. Para isso, podemos fazer o produto vetorial usando os vetores diretores das retas, e o resultado deve ser igual a .

2º passo

Em seguida, uma vez que os vetores diretores das retas são LD, precisamos verificar se as retas são distintas, para isso usamos um ponto qualquer pertencente a uma das retas na equação da outra reta e analisamos se ele não pertence a esta. Se o ponto não pertencer a ambas as retas, então são retas paralelas distintas.
