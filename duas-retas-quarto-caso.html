Posição relativa

Neste caso, as retas não possuem um ponto de intersecção e estão em planos distintos.

Como identificar

1º passo

Para identificar se duas retas são reversas, precisamos primeiro verificar se os vetores diretores das retas são linearmente independentes (LI), pois, neste caso, as retas possuem direções distintas. Para isso, devemos fazer o produto vetorial usando os vetores diretores das retas, e o resultado deve ser diferente de .

2º passo

Em seguida, uma vez que os vetores diretores são LI, devemos verificar se as retas não pertencem ao mesmo plano, para isso precisamos obter um vetor a partir de um ponto de cada reta que temos e fazer o produto misto entre os três vetores (um da reta r, um da reta s e o terceiro obtido), se o resultado for diferente de 0, então as retas são reversas.
