Posição relativa entre duas retas

Na geometria há quatro possibilidades para duas retas no espaço, são elas: paralelas coincidentes (iguais), paralelas distintas, concorrentes ou reversas..

A seguir iremos ver esses casos e como podemos identificá-los algebricamente.

Posição relativa

Neste caso, as retas se cruzam em um ponto.

Como identificar

1º passo

Para identificar se duas retas são concorrentes, primeiro precisamos verificar se os vetores diretores das retas são linearmente independentes (LI), pois, neste caso, as retas possuem direções distintas. Para isso, devemos fazer o produto vetorial usando os vetores diretores das retas, e o resultado deve ser diferente de .

2º passo

Em seguida, uma vez que os vetores diretores são LI, devemos verificar se as retas pertencem ao mesmo plano, neste caso elas se cruzam, e para isso precisamos obter um vetor a partir de um ponto de cada reta que temos e fazer o produto misto entre os três vetores (um da reta r, um da reta s e o terceiro obtido), se o resultado for igual a 0, então as retas r e s são coplanares e, portanto, são concorrentes.
