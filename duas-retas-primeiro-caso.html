Posição relativa entre duas retas

Na geometria há quatro possibilidades para duas retas no espaço, são elas: paralelas coincidentes (iguais), paralelas distintas, concorrentes ou reversas.

A seguir iremos ver esses casos e como podemos identificá-los algebricamente.

Posição relativa

Neste caso, as retas são iguais, logo, ocupam o mesmo lugar no espaço.

Como identificar

1º passo

Para identificar se duas retas são paralelas coincidentes, primeiro precisamos verificar se os vetores diretores das retas são linearmente dependentes (LD), pois, neste caso, ambas as retas possuem a mesma direção no espaço, sendo assim, são retas paralelas. Para isso, podemos fazer o produto vetorial usando os vetores diretores das retas, e o resultado deve ser igual a .

2º passo

Em seguida, uma vez que os vetores diretores das retas são LD, precisamos verificar se as retas são iguais, para isso usamos um ponto qualquer pertencente a uma das retas na equação da outra reta e analisamos se ele também pertence a esta. Se o ponto pertence às duas retas, então são retas paralelas coincidentes.
